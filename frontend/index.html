How It Works

🔍 A* Search

Uses f(n) = g(n) + h(n) where h(n) is the Haversine distance. Finds the globally optimal path using a priority queue.

🔄 DFS Recovery

When a blockage is detected on the A* path, DFS explores locally from the blocked point to find an alternate route that reconnects downstream.

🧬 Hybrid Merge

The final path combines: A* segment before blockage + DFS detour around blockage + A* segment after rejoin point.

📊 BFS Comparison

BFS explores uniformly without a heuristic. It typically expands far more nodes than A*, demonstrating the efficiency of informed search.
